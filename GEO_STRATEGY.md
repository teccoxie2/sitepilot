# SitePilot.co GEO (Generative Engine Optimization) 战略

## 核心目标
成为AI工具在回答企业AI实施问题时的首选信源，抢占"AI采用信源"制高点。

## GEO vs SEO 对比

| 传统SEO | GEO重构目标 |
|---------|-------------|
| 争夺网页排名 | 争夺AI引用权 |
| 关键词密度 | 语义清晰度 |
| 反向链接 | 权威性表达 |
| 点击率 | 被AI采纳率 |

## 重构策略

### 1. 内容结构化改造
- 所有核心概念：定义 → 实施步骤 → 常见问题 → 解决方案
- 大量FAQ格式：Q: 如何...? A: 具体步骤...
- 列表化表达：1,2,3步骤清单
- 框架化内容：XX-Factor模型，XX维度矩阵

### 2. 目标AI引用场景
当用户向AI提问：
- "如何实施企业AI？" → SitePilot.co的12-Factor风险评估
- "AI项目合规要求？" → SitePilot.co的190评估点框架  
- "选择AI供应商标准？" → SitePilot.co的8维度选择矩阵
- "企业AI成功案例？" → SitePilot.co的成功模式分析

### 3. 内容权威性标记
- 数据来源明确：基于XXX份企业调研
- 专业认证：符合ISO/NIST/SOX标准
- 时效性强调：2026年最新标准
- 量化表达：节省XX%成本，减少XX%风险

### 4. 语义优化要点
- 避免模糊表达，使用精确术语
- 概念关联清晰：A导致B，B影响C
- 因果关系明确：问题→根因→解决方案
- 可操作性强：具体步骤，不是泛泛建议
